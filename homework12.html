Download the following two files and store them in the same directory.

  • http://cs.harding.edu/gfoust/classes/comp431/gibberish.html
  • https://code.jquery.com/jquery-3.3.1.min.js

Look at the HTML source. Notice that it includes the jQuery JavaScript file, and that it assumes it is in the same directory as the HTML file. It also incldes a JavaScript file (from the same directory) named gibberish.js. This is the file you are to write.

The code you will write should simply select nodes from the DOM using jQuery and add the proper classes to them using the addClass method. You may not make any changes to the HTML file.

Problems

  1. All <p> tags should have class text.
  2. All <li> tags inside an <ul> should have class spaced.
  3. Any tag whose class is important should have the class highlighted.
  4. The <div> tags which contain <img> tags should have the class bordered.
  5. Any tags which are in the same <div> as an <img> should have the class centered.
  6. The first <div> that contains an <img> should have class right.
  7. The last <div> that contains an <img> should have class left.
  8. The even numbered <li> tags (number 2, number 4, ...) in an <ol> tag should have the class distinct.
  9. The tag whose id is key should have the class blue.
  10. The tag whose id is key is in a <p> tag inside a <div>. Any <em> tags in that same div should have the class bold.
